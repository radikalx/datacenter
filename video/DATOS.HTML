autoplay

reproduce el video ni bien se abra la web, sin necesidad de dar clic en el boton de reproduccion

loop

este atributo se encarga de reproducir el audio siempre q termine

preload

este atributo tiene un comportamiento especial cuando estamos trabajando con archivos multimedia q son muy pesados nuestro sitio web tienede a ponerse lento este atributo almacena de manera temporal para q sea mas rapido el acceso a ellos

  • none : no almacenar nuestros archivos
  • auto : almacena temporalmente el archivo multimedia
